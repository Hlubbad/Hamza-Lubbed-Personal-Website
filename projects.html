Overview

Course: SITE 1101 ‚Äî Introduction to Digital Logic

Project Title: Logic Gates Construction

Objective

Design, build and verify the behaviour of fundamental logic gates (AND, OR, NOT, NAND, NOR, XOR) using discrete electronic components on a breadboard. Demonstrate correct truth-table outputs for all gate combinations and explain gate implementation choices.

Materials

  • Breadboard and jumper wires
  • TTL logic ICs (74xx family) and/or discrete transistors & resistors
  • LEDs and current-limiting resistors for output indicators
  • 5V DC power supply
  • Multimeter and logic probe/oscilloscope (optional)

Methodology

  1. Select implementation style (IC-based or transistor-level).
  2. Create schematic diagrams for each gate and plan breadboard layout.
  3. Assemble gates on the breadboard, wire inputs to test switches and outputs to LEDs.
  4. Systematically test all input combinations and record outputs.
  5. Compare measured outputs to expected truth tables and note discrepancies.

Results & Learning Outcomes

All primary gates (AND, OR, NOT) produced the expected outputs when tested across input combinations. Derivative gates (NAND, NOR, XOR) were implemented and verified. The exercise reinforced practical wiring discipline, propagation delay awareness, and the importance of proper power decoupling and input pull-down/up resistors.
